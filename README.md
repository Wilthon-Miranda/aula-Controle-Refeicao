# Dieta App

Este é um aplicativo de controle de refeições, desenvolvido em React Native com Expo, para ajudar usuários a monitorar sua alimentação e manter o foco na dieta.

## 💡 Ideia do Projeto
O objetivo do Dieta App é permitir que o usuário registre suas refeições diárias, classificando cada uma como dentro ou fora da dieta. O app exibe estatísticas, histórico de refeições e incentiva o acompanhamento dos hábitos alimentares.

## 🚀 Como rodar o projeto

1. **Pré-requisitos:**
	- Node.js instalado
	- Expo CLI instalado globalmente (`npm install -g expo-cli`)
	- Emulador Android/iOS ou dispositivo físico

2. **Instale as dependências:**
	```bash
	npm install
	```

3. **Inicie o projeto:**
	```bash
	npm start
	```
	Ou use:
	```bash
	expo start
	```

4. **Execute no dispositivo ou emulador:**
	- Siga as instruções do Expo para rodar no Android, iOS ou Web.

## 📱 Funcionalidades
- Cadastro de novas refeições com nome, descrição, data, hora e status (dentro/fora da dieta)
- Listagem das refeições por data
- Estatísticas de desempenho na dieta
- Feedback visual após cadastrar refeições
